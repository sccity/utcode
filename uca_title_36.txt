Utah Code Annotated § 36-1-101.1 Definitions.
---------------------------------------------
As used in this part:Utah Code Annotated § 36-1-101.1(1): "Census block" means any one of the 71,207 individual geographic areas into which the Bureau of the Census of the United States Department of Commerce has divided the state of Utah, to each of which the Bureau of the Census has attached a discrete population tabulation from the 2020 decennial census.
Utah Code Annotated § 36-1-101.1(2): "Senate block equivalency file" means the electronic file designated as SB2006S02_BEF.txt that assigns each of Utah's 71,207 census blocks to a particular Utah State Senate district.
Utah Code Annotated § 36-1-101.1(3): "Senate shapefile" means the electronic shapefile that:
Utah Code Annotated § 36-1-101.1(3)(a): is the resulting projection of the Senate block equivalency file; and
Utah Code Annotated § 36-1-101.1(3)(b): stores the boundary of each of the 29 Utah State Senate districts.
Utah Code Annotated § 36-1-101.1(4): "Shapefile" means the digital vector storage format for storing geometric location and associated attribute information.


Utah Code Annotated § 36-1-101.5 Utah State Senate -- District boundaries.
--------------------------------------------------------------------------
Utah Code Annotated § 36-1-101.5(1): The Utah State Senate shall consist of 29 members, with one member to be elected from each Utah State Senate district.
Utah Code Annotated § 36-1-101.5(2): The Legislature adopts the official census population figures and maps of the Bureau of the Census of the United States Department of Commerce developed in connection with the taking of the 2020 national decennial census as the official data for establishing Senate district boundaries.
Utah Code Annotated § 36-1-101.5(3): 
Utah Code Annotated § 36-1-101.5(3)(a): The Legislature enacts the district numbers and boundaries of the Senate districts designated in the Senate block equivalency file and resulting Senate shapefile that is the electronic component of Laws of Utah 2021, Second Special Session,
Utah Code Annotated § 36-1-101.5(3)(a)(i): for purposes of nominating and electing certain members of the Utah State Senate beginning January 1, 2022; and
Utah Code Annotated § 36-1-101.5(3)(a)(ii): for all other purposes beginning January 1, 2023.
Utah Code Annotated § 36-1-101.5(3)(b): The Legislature shall ensure that the Senate shapefile, and the Senate district boundaries generated from the Senate shapefile, are accessible on the Utah Legislature's website.

A person may not employ or solicit another to serve as a lobbyist for compensation contingent in whole or part upon the passage, defeat, or amendment of legislative action or the approval, modification, or denial of a certain executive action.

Utah Code Annotated § 36-12-6 Permanent committees -- House and Senate management -- Members -- Chair -- Legislative Management Committee -- Membership -- Chair and vice-chair -- Meetings -- Quorum.
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Utah Code Annotated § 36-12-6(1): There are hereby established as permanent committees of the Legislature a House Management Committee and a Senate Management Committee.  The House Management Committee shall consist of eight members of the House of Representatives, four from each major political party. Utah Code Annotated § 36-35-104 Review of court rules -- Criteria.
------------------------------------------------------------------
Utah Code Annotated § 36-35-104(1): As used in this section, "court rule" means a proposal for a court rule, a new court rule, or an existing court rule.
Utah Code Annotated § 36-35-104(2): The committee may review and evaluate:
Utah Code Annotated § 36-35-104(2)(a): a submission of:
Utah Code Annotated § 36-35-104(2)(a)(i): a new court rule; or
Utah Code Annotated § 36-35-104(2)(a)(ii): a proposal for a court rule; and
Utah Code Annotated § 36-35-104(2)(b): an existing court rule.
Utah Code Annotated § 36-35-104(3): If the committee chooses to conduct a review of a court rule as provided under Subsection (2), the review shall be based on the following criteria:
Utah Code Annotated § 36-35-104(3)(a): whether the court rule is authorized by the state constitution or by statute;
Utah Code Annotated § 36-35-104(3)(b): if authorized by statute, whether the court rule complies with legislative intent;
Utah Code Annotated § 36-35-104(3)(c): whether the court rule is in conflict with existing statute or governs a policy expressed in statute;
Utah Code Annotated § 36-35-104(3)(d): whether the court rule is primarily substantive or procedural in nature;
Utah Code Annotated § 36-35-104(3)(e): whether the court rule infringes on the powers of the executive or legislative branch of government;
Utah Code Annotated § 36-35-104(3)(f): the impact of the court rule on an affected person;
Utah Code Annotated § 36-35-104(3)(g): the purpose for the court rule, and if applicable, the reason for a change to an existing court rule;
Utah Code Annotated § 36-35-104(3)(h): the anticipated cost or savings due to the court rule to:
Utah Code Annotated § 36-35-104(3)(h)(i): the state budget;
Utah Code Annotated § 36-35-104(3)(h)(ii): local governments; and
Utah Code Annotated § 36-35-104(3)(h)(iii): individuals; and
Utah Code Annotated § 36-35-104(3)(i): the cost to an affected person of complying with the court rule.
