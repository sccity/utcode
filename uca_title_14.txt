Utah Code Annotated § 14-2-1 Definitions -- Payment bond required -- Right of action -- Attorney fees.
------------------------------------------------------------------------------------------------------
Utah Code Annotated § 14-2-1(1): For purposes of this chapter:
Utah Code Annotated § 14-2-1(1)(a): "Commercial contract" means a contract for the construction, alteration, or repair of the following if it is not residential construction:
Utah Code Annotated § 14-2-1(1)(a)(i): a building;
Utah Code Annotated § 14-2-1(1)(a)(ii): a structure; or
Utah Code Annotated § 14-2-1(1)(a)(iii): an improvement upon land that is not associated with a single family detached housing.
Utah Code Annotated § 14-2-1(1)(b): "Contractor" means any person who is or may be awarded an original commercial contract for the construction, alteration, or repair of any building, structure, or improvement upon land.
Utah Code Annotated § 14-2-1(1)(c): "Owner" means any person contracting with the original contractor for construction, alteration, or repair of the following if it is not residential construction:
Utah Code Annotated § 14-2-1(1)(c)(i): a building;
Utah Code Annotated § 14-2-1(1)(c)(ii): a structure; or
Utah Code Annotated § 14-2-1(1)(c)(iii): an improvement upon land.
Utah Code Annotated § 14-2-1(1)(d): 
Utah Code Annotated § 14-2-1(1)(d)(i): "Residential construction" means the construction, alteration, or repair of:
Utah Code Annotated § 14-2-1(1)(d)(i)(A): single family detached housing; or
Utah Code Annotated § 14-2-1(1)(d)(i)(B): multifamily attached housing up to and including a fourplex.
Utah Code Annotated § 14-2-1(1)(d)(ii): "Residential construction" includes rental housing.
Utah Code Annotated § 14-2-1(2): (3) Before any original commercial contract exceeding $50,000 in amount for the construction, alteration, or repair of any building, structure, or improvement upon land is awarded to any contractor, the owner shall obtain from the contractor a payment bond:
Utah Code Annotated § 14-2-1(2)(a): (3) complying with Subsection
Utah Code Annotated § 14-2-1(2)(b): that becomes binding upon the award of the original commercial contract to the contractor.
Utah Code Annotated § 14-2-1(3): The payment bond shall be:
Utah Code Annotated § 14-2-1(3)(a): with a surety or sureties satisfactory to the owner for the protection of all persons supplying labor, services, equipment, or material in the prosecution of the work provided for in the commercial contract; and
Utah Code Annotated § 14-2-1(3)(b): in a sum equal to the original commercial contract price.
Utah Code Annotated § 14-2-1(4): A person shall have a right of action on a payment bond under this chapter for any unpaid amount due that person if that person:
Utah Code Annotated § 14-2-1(4)(a): has furnished labor, services, equipment, or material in the prosecution of the work provided for in the commercial contract for which the payment bond is furnished under this chapter; and
Utah Code Annotated § 14-2-1(4)(b): has not been paid in full within 90 days after the last day on which that person:
Utah Code Annotated § 14-2-1(4)(b)(i): performed the labor or service for which a claim is made; or
Utah Code Annotated § 14-2-1(4)(b)(ii): supplied the equipment or material for which the claim is made.
Utah Code Annotated § 14-2-1(5): 
Utah Code Annotated § 14-2-1(5)(a): An action under this section shall be brought in a court of competent jurisdiction in the county where the commercial contract was to be performed and not elsewhere.
Utah Code Annotated § 14-2-1(5)(b): An action under this section is barred if not commenced within one year after the last day on which the claimant:
Utah Code Annotated § 14-2-1(5)(b)(i): performed the labor or service on which the claim is based; or
Utah Code Annotated § 14-2-1(5)(b)(ii): supplied the equipment or material on which the claim is based.
Utah Code Annotated § 14-2-1(5)(c): The obligee named in the payment bond need not be joined as a party to an action under this section.
Utah Code Annotated § 14-2-1(5)(d): In any action upon a payment bond under this section, the court may award reasonable attorney fees to the prevailing party, which attorney fees shall be taxed as costs in the action.
Utah Code Annotated § 14-2-1(6): The payment bond shall be exhibited to any interested person upon request.
Utah Code Annotated § 14-2-1(7): In any suit upon a payment bond under this chapter, the court shall award reasonable attorney fees to the prevailing party.
